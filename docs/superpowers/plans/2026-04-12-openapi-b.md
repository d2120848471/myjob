# OpenAPI B Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在本地和测试环境为当前 GoFrame 后台接入原生 OpenAPI 文档能力，同时保持现有接口 `code / msg / data` 返回契约不变。

**Architecture:** 逐步把接口迁移到 GoFrame 原生可生成 OpenAPI 的 req/res + g.Meta 形态；运行时继续保留项目自定义响应输出层，不接管为 GoFrame 默认 `message` 风格；通过环境开关只在开发和测试开放文档入口。

**Tech Stack:** Go, GoFrame v2.10, OpenAPI v3, Redoc/Swagger page, Go contract tests.
